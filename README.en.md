## ğŸš€ Getting Started

### Prerequisites

- Node.js 18+
- pnpm 8+

### Installation

```bash
# Clone the repository
git clone https://github.com/yourusername/portfolio.git

# Navigate to project directory
cd portfolio

# Install dependencies
pnpm install

# Start development server
pnpm dev
```

### Build for Production

```bash
# Generate static site
pnpm generate

# Or build for SSR
pnpm build
```
